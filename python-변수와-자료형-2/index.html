Python 변수와 자료형(2)

Tuple

  • 튜플 생성

    • 소괄호로 묶어서 생성할 수도 있고
    • 소괄호를 생략하고 콤마로만 구분해서 생성할 수도 있다.
    • 튜플의 원소가 하나일 경우, 반드시 마지막에 콤마를 찍어야 한다.
    • tuple() 생성자를 사용할 수도 있다.
    1tup1 = (1,2,3)
    2tup2 = 1,2,3
    3tup3_1 = 1
    4tup3_2 = 1,
    5tup4 = tuple("abcde")
    6
    7print(type(tup1))   # <class 'tuple'>
    8print(type(tup2))   # <class 'tuple'>
    9print(type(tup3_1)) # <class 'int'>
    10print(type(tup3_2)) # <class 'tuple'>
    11print(tup4)         # ('a', 'b', 'c', 'd', 'e')

  • 튜플의 특징

    • 튜플을 포함한 모든 자료형이 튜플에 포함될 수 있다.
    • 순서가 있다.
    • 한 번 생성되면 값을 변경할 수 없다.
      • 그러나 + 연산, * 연산은 가능하다.
    • 튜플을 활용하면 여러 값을 한 번에 각기 다른 변수에 할당하는 것이 가능하다.
      • 상기했듯 굳이 ()로 묶지 않아도 콤마로 구분만 되어 있으면 튜플이 생성된다.
    • 튜플의 해제 할당 기능을 사용하면 두 변수의 값을 바꾸는 것도 가능하다.
    1# 순서가 있으므로 인덱스를 통해 접근이 가능하다.
    2tup = ('one','two', 'three')
    3print(tup[0:2])    # ('one', 'two')
    4
    5# 값을 변경할 수 없다.
    6tup[0] = 'zero'   # 'tuple' object does not support item assignment
    7
    8# +, * 연산은 가능하다.
    9tup1 = (1,2,3)
    10tup2 = (4,5,6)
    11print(tup1+tup2)  # (1, 2, 3, 4, 5, 6)
    12print(tup1*2)     # (1, 2, 3, 1, 2, 3)
    13
    14# 여러 값을 변수에 한 번에 할당하는 것이 가능하다.
    15email,phone = "email@email.com", "010-1234-5678"
    16print(email,phone)   # email@email.com 010-1234-5678
    17
    18# 두 변수의 값을 바꾸는 것도 가능하다.
    19email, phone = phone, email
    20print(email,phone)   # 010-1234-5678 email@email.com


List

  • 리스트 생성

    • list() 생성자를 사용하여 생성이 가능하다.
    • [] 로 생성이 가능하다.
    1lst1 = list()
    2lst2 = []
    3print(type(lst1))  # <class 'list'>
    4print(type(lst2))  # <class 'list'>

  • 리스트 특징

    • 리스트를 포함한 모든 자료형이 리스트에 포함될 수 있다.
    • 순서가 있다.
    • 한 번 생성된 값을 변경 가능하다.
    1# 순서가 있으므로 인덱스를 통해 접근이 가능하다.
    2lst = ['one', 'two', 'three']
    3print(lst[0:2])   # ['one', 'two']
    4
    5# 변경이 가능하다.
    6lst[0] = 'zero'
    7print(lst)        # ['zero', 'two', 'three']

  • 리스트 관련 메서드

    아래 메서드들 중 일부는 튜플, 문자열 등에서도 사용 가능하다.

    • len(): 요소의 개수를 반환
    1lst = [1,2,3]
    2print(len(lst))   # 3
    • .append(): 요소를 추가
    1lst = [1,2,3]
    2lst.append(4)
    3print(lst)   #[1, 2, 3, 4]
    • .insert(인덱스, 추가할 값): 지정한 인덱스에 값을 삽입
    1lst = [1,2,3]
    2lst.insert(1,4)
    3print(lst)   # [1, 4, 2, 3]
    • .remove(): 요소를 삭제
    1lst = [1,2,3]
    2lst.remove(1)
    3print(lst)  # [2,3]
    • del: 요소를 삭제
      • remove와의 차이는 remove는 삭제할 값을 지정하지만 del은 삭제할 인덱스를 지정한다는 것이다.
    1lst = [1,2,3]
    2del lst[1]
    3print(lst)  # [1,3]
    • .pop(): 리스트의 마지막 요소를 삭제 후 반환
    1lst = [1,2,3]
    2# 삭제 후 반환한다.
    3print(lst.pop())  # 3
    4print(lst)        # [1,2]
    • .clear(): 모든 요소 삭제
    1lst = [1,2,3]
    2lst.clear()
    3print(lst)      # []
    • .reverse(): 순서를 뒤집는다.
    1lst = [1,2,3]
    2lst.reverse()
    3print(lst)    # [3, 2, 1]
    • .sort(): 정렬한다.
    1lst = [9,3,6]
    2lst.sort()
    3print(lst)    # [3,6,9]
    • sorted(): 정렬한다.
      • .sort()와의 차이는 원본 배열을 변경하지 않는다는 것이다.
    1lst = [9,3,6]
    2print(sorted(lst))  # [3,6,9]
    3print(lst)          # [9,3,6]
    • .copy()
      • 복사에는 얕은 복사와 깊은 복사가 있는데 .copy()를 사용하면 깊은 복사가 가능하다.
      • [:]를 활용해도 깊은 복사가 가능하다.
    1# 얕은 복사
    2lst1 = [1,2,3]
    3lst2 = lst1  
    4lst2[0] = 9
    5# 얕은 복사이기에 같은 객체를 가리키고 있어 하나를 변경하면 둘 다 변경된다.
    6print(lst1,lst2)  # [9, 2, 3] [9, 2, 3]
    7
    8# 깊은 복사
    9lst1 = [1,2,3]
    10lst2 = lst1.copy()
    11lst2[0] = 9
    12# 깊은 복사이기에 서로 다른 객체를 가리키고 있어 하나를 변경해도 다른 하나는 변경되지 않는다.
    13print(lst1,lst2)  # [1, 2, 3] [9, 2, 3]
    14
    15# [:]를 활용한 깊은 복사
    16lst1 = [1,2,3]
    17lst2 = lst1[:]
    18lst2[0] = 9
    19print(lst1,lst2)  # [1, 2, 3] [9, 2, 3]


Dictionary

  • 생성

    • dict()생성자를 사용해 생성 가능하다.
    • {}를 사용해 생성 가능하다.
    1dict1 = dict()
    2dict2 = {}
    3print(type(dict1))  # <class 'dict'>
    4print(type(dict2))  # <class 'dict'>

  • 딕셔너리의 특징

    • 키(key)-값(value) 쌍을 요소로 갖는 자료형이다.
    • 키는 중복이 불가능하다.
      • 키를 중복으로 사용할 경우 하나의 키를 제외한 모든 중복된 키는 무시된다.
    • 키는 변경 immutable 타입이어야 하며 값은 immutable과 mutable 모두 가능하다.
      • 변경 불가능한 문자열이나 튜플 등은 키가 될 수 있다.
      • 변경 가능한 리스트는 키가 될 수 없다.
      • 값에는 딕셔너리를 포함한 모든 자료형이 올 수 있다.
    • 순서가 없는 자료형으로 key를 통해 값에 접근해야 한다.
    • 이미 입력된 값의 변경이 가능하다.
    • 요소의 추가와 삭제가 가능하다.
    1# key에는 변경 불가능한 자료형이, 값에는 모든 자료형이 올 수 있다.
    2my_dict = {"취미":['축구','야구'],"이름":'홍길동',"나이":28,"가족":{"엄마":"김엄마","아빠":"홍아빠"}}
    3
    4# key를 통해 값에 접근할 수 있다.
    5print(my_dict['이름'])      # 홍길동
    6
    7# 변경이 가능하다.
    8my_dict['나이']=14
    9print(my_dict['나이'])      # 14 
    10
    11# 요소(키-값) 추가
    12my_dict["email"] = "email@email.com"
    13print(my_dict["email"])    # email@email.com
    14
    15
    16# 요소(키-값) 삭제
    17del my_dict['email']
    18print(my_dict)  # {'취미': ['축구', '야구'], '이름': '홍길동', '나이': 14, '가족': {'엄마': '김엄마', '아빠': '홍아빠'}}

  • 딕셔너리 관련 함수

    • .keys(): key를 리스트로 반환한다.
    1my_dict = {"취미":['축구','야구'],"이름":'홍길동',"나이":28}
    2print(my_dict.keys())  # dict_keys(['취미', '이름', '나이'])
    • .values(): 값을 리스트로 반환한다.
    1my_dict = {"취미":['축구','야구'],"이름":'홍길동',"나이":28}
    2print(my_dict.values())  # dict_values([['축구', '야구'], '홍길동', 28])
    • .items(): 키-값 쌍을 리스트로 반환한다.
    1my_dict = {"취미":['축구','야구'],"이름":'홍길동',"나이":28}
    2print(my_dict.items()) # dict_items([('취미', ['축구', '야구']), ('이름', '홍길동'), ('나이', 28)])
    • .clear(): 모든 요소 삭제
    1my_dict = {"취미":['축구','야구'],"이름":'홍길동',"나이":28}
    2my_dict.clear()
    3print(my_dict)  # {}
    • .get(): 키로 값 얻기
      • 그냥 키로만 접근하는 것과의 차이는 존재하지 않는 키로 접근할 경우, 키로만 접근하면 error가 발생하지만, .get()은 None을 반환한다는 것이다.
    1my_dict = {"취미":['축구','야구'],"이름":'홍길동',"나이":28}
    2print(my_dict['email'])      # KeyError: 'email'
    3print(my_dict.get('email'))  # None


Set

  • 생성

    • 다른 자료형들과 달리 set은 set() 생성자로만 생성할 수 있다.
    • 반드시 문자열 또는 괄호로 묶어줘야 한다(괄호의 종류는 상관 없다).
    • 비어있는 자료형도 생성 가능하다.
    1my_set1 = set({1,2,3})
    2my_set2 = set("Hello!")
    3my_set3 = set()
    4
    5print(type(my_set1))  # <class 'set'>
    6print(type(my_set2))  # <class 'set'>
    7print(my_set1)        # {1, 2, 3}
    8print(my_set2)        # {'l', '!', 'o', 'e', 'H'}
    9print(my_set3)        # set()

  • Set(집합) 자료형의 특징

    • 중복을 허용하지 않는다.
    • 순서가 없다.
    1my_set = set("Hello!")
    2
    3print(my_set2)        # {'l', '!', 'o', 'e', 'H'}
    4# 중복을 허용하지 않기에 두 번 들어간 l은 하나만 들어가게 된다.
    5# 순서가 없기에 순서대로 들어가지 않는다.

  • 교집합, 합집합, 차집합 구하기

    1my_set1 = set([1,2,3])
    2my_set2 = set([3,4,5])
    3
    4# 교집합
    5print(my_set1 & my_set2)                # {3}
    6print(my_set1.intersection(my_set2))    # {3}
    7
    8# 합집합
    9print(my_set1 | my_set2)                # {1, 2, 3, 4, 5}
    10print(my_set1.union(my_set2))           # {1, 2, 3, 4, 5}
    11
    12# 차집합
    13print(my_set1-my_set2)                  # {1, 2}
    14print(my_set1.difference(my_set2))      # {1, 2}
    15print(my_set2-my_set1)                  # {4, 5}
    16print(my_set2.difference(my_set1))      # {4, 5}

  • 집합 관련 함수들

    • .add(): 요소를 1개 추가한다.
    1my_set = set([1,2,3])
    2my_set.add(4)
    3print(my_set)   # {1, 2, 3, 4}
    • .update(): 요소 여러 개 추가하기
    1my_set = set([1,2,3])
    2my_set.update([4,5,6])
    3print(my_set)   # {1, 2, 3, 4, 5, 6}
    • .remove(): 특정 요소 제거하기
    1my_set = set([1,2,3])
    2my_set.remove(2)
    3print(my_set)   # {1, 3}




자료형의 변경

  • str(), int(), float(), bool() 등의 함수를 사용해서 변경하면 된다.

  • 주의점

    • int()의 경우 숫자가 아닌 것을 숫자로 변환할 수 없으며, 소수점이 있는 숫자 형식의 문자열을 정수형으로 변환할 수 없다.
    • 소수점이 없는 숫자 형식의 문자열은 정수형으로 변환이 가능하다.
    • 소수점이 있든 없든 숫자 형식이기만 하면 실수형으로 변환이 가능하다.
    • 변환 함수는 변환된 값을 반환할 뿐 실제로 자료형을 변경시키는 것은 아니다.
    1# 숫자 형식인 문자열의 정수와 실수 변환
    2chr = "1"
    3print(type(int(chr)))       # <class 'int'>
    4print(type(float(chr)))     # <class 'float'>
    5
    6# 변환 함수는 변환된 값을 반환만 할 뿐이다.
    7var = True
    8print(type(str(var)))   # <class 'str'>
    9print(type(var))        # <class 'bool'>
    10
    11# 아래와 같이 재할당 해주거나 다른 변수에 담아서 사용해야 한다.
    12var = True
    13var = str(var)
    14print(type(var))        # <class 'str'>
