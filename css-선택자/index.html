CSS 선택자

11.01.2021 — CSS, web — 2 min read

선택자

  • 선택자는 스타일을 적용하고자 하는 HTML 요소를 선택하기 위한 수단이다.

  • 선택자로 HTML 요소를 선택하고 {}내에 속성과 값을 지정하여 다양한 스타일을 설정할 수 있다. 여러 개의 속성을 연속해서 지정할 수 있으며 ;으로 구분한다. 복수개의 셀렉터를 연속하여 지정할 수 있으며 쉼표( , )로 구분한다.

    1h1{color:red;font-size:12px;}
    2선택자: h1
    3선언: color:red;, font-size:12px;
    4선언 블록: {color:red;font-size:12px;}
    5속성(프로퍼티):color, font-size
    6값: red, 12px
    7
    8/*복수의 선택자 지정*/
    9h1, p { color: red; }

  • 타입 선택자(기초 선택자)

    • 가장 쉽고 기본이 되는 기초적인 선택자

    • HTML요소명, 요소의 클래스 속성 값, ID 값

      • HTML선택자(태그 셀렉터): HTML요소가 선택자인 선택자

        1<!--형식: 태그명-->
        2<!DOCTYPE html>
        3<html>
        4<head>
        5  <style>
        6    /* 모든 p 태그 요소를 선택 */
        7    p { color: red; }
        8  </style>
        9</head>
        10<body>
        11  <h1>Heading</h1>
        12  <div>
        13    <p>paragraph 1</p> p태그 요소 이므로 red로 출력
        14    <p>paragraph 2</p> p태그 요소 이므로 red로 출력
        15  </div>
        16  <p>paragraph 3</p>   p태그 요소 이므로 red로 출력
        17</body>
        18</html>

      • 클래스 선택자: HTML 클래스 속성 값, 요소들을 원하는 그룹으로 묶을 수 있음, HTML 문서 내의 동일한 클래스 값을 가지는 모든 요소에 적용, .(온점)으로 표시한다.

        1<!--형식: .class 어트리뷰트 값-->
        2<!DOCTYPE html>
        3<html>
        4    <head>
        5          <style>
        6            /* class 어트리뷰트 값이 text-center인 모든 요소를 선택 */
        7            .text-center { text-align: center; }
        8            /* class 어트리뷰트 값이 text-large인 모든 요소를 선택 */
        9            .text-large  { font-size: 200%; }
        10            /* class 어트리뷰트 값이 text-red인 모든 요소를 선택 */
        11            .text-red    { color: red; }
        12            /* class 어트리뷰트 값이 text-blue인 모든 요소를 선택 */
        13            .text-blue   { color: blue; }
        14          </style>
        15    </head>
        16    <body>
        17          <!--class 어트리뷰트 값이 text-center이므로 중앙에 표시됨-->
        18          <p class="text-center">Center</p>
        19          <!--class 어트리뷰트 값이 text-red이므로 빨강으로 표시됨-->
        20          <!--class 어트리뷰트 값이 text-large이므로 200% 크기도 표시됨-->
        21          <p class="text-large text-red">Large Red</p>
        22          <!--class 어트리뷰트 값이 text-center, text-large, text-blue이므로
        23            중앙에 빨강색으로 200%크기로 표시됨-->
        24          <p class="text-center text-large text-blue">Center Large Blue</p>
        25    </body>
        26</html>

      • ID 선택자: HTML ID 속성 값에 CSS를 적용, HTML ID 속성은 HTML 문서 내의 유일한 값으로 고유의 요소를 식별한다. 문서 내의 같은 ID를 가진 유일한 요소에 적용한다. #으로 표현한다.

        1<!--형식: #id 어트리뷰트 값-->
        2<!DOCTYPE html>
        3<html>
        4   <head>
        5     <style>
        6       /* id 어트리뷰트 값이 p1인 요소를 선택 */
        7       #p1 { color: red; }
        8     </style>
        9   </head>
        10   <body>
        11     <h1>Heading</h1>
        12     <div class="container">
        13       <p id="p1">paragraph 1</p> <!--id 어트리뷰트 값이 p1이므로 빨강으로 출력-->
        14       <p id="p2">paragraph 2</p>
        15     </div>
        16       <p>paragraph 3</p>
        17        </body>
        18</html>

      • 그룹 지정: 여러 요소에 동일한 CSS 스타일 적용

      • 전체 셀렉터

        1<!--형식: * -->
        2<!DOCTYPE html>
        3<html>
        4    <head>
        5      <style>
        6        /* 모든 요소를 선택 */
        7        * { color: red; }
        8      </style>
        9    </head>
        10    <body>
        11      <h1>Heading</h1>
        12      <div>
        13        <p>paragraph 1</p>  <!--전체 셀렉터이므로 전부 빨강으로 출력-->
        14        <p>paragraph 2</p>  <!--전체 셀렉터이므로 전부 빨강으로 출력-->
        15      </div>
        16      <p>paragraph 3</p>    <!--전체 셀렉터이므로 전부 빨강으로 출력-->
        17   </body>
        18</html>

  • 고급 선택자

    • HTML 문서의 요소들의 계층관계를 이용하여 선택하는 선택자

    • CSS는 상속을 통해 부모 요소의 속성을 자식에게 상속한다.

      • 상속 되는 것: Text 관련 요소(font, color 등)

      • 상속 되지 않는 것: Box model 관련 요소(width, height 등)와 position관련 요소(position, left, right 등)

      • MDN에서 상속 여부를 확인 할 수 있다.

    • HTML 요소들의 계층관계: 요소들 간의 포함 관계에 따라 자손요소, 직계 자손 요소, 형제 요소, 인접 형제 요소등으로 나뉜다.

      • 자손 요소: 특정 요소의 내부에 포함된 요소

      • 직계 자손 요소: 특정 요소의 바로 아래 있는 요소

      • 형제 요소: 같은 계층 단계에 있는 요소

      • 인접 형제 요소: 바로 아래 있는 형제 요소

      • 고급 선택자의 종류

        -하위 선택자: 어떤 요소 하위에 있는 특정 자손 요소 선택 시 사용. 대괄호로 표현

        -형제 선택자: 특정 요소 다음에 나오는 형제 관계 요소들 선택. ~(물결무늬)로 표현

        1<!DOCTYPE html>
        2<html>
        3  <head>
        4      <style>
        5        /* p 요소의 형제 요소 중에 p 요소 뒤에 위치하는 ul 요소를 모두 선택한다.*/
        6        p ~ ul { color: red; }
        7      </style>
        8  </head>
        9  <body>
        10      <div>A div element.</div>
        11      <ul>
        12        <li>Coffee</li>
        13        <li>Tea</li>
        14        <li>Milk</li>
        15      </ul>
        16    
        17      <p>The first paragraph.</p>
        18      <ul>
        19        <li>Coffee</li> <!--p 요소의 형제 요소 중에 p 요소 뒤에 위치하는 ul 요소-->
        20        <li>Tea</li>    <!--p 요소의 형제 요소 중에 p 요소 뒤에 위치하는 ul 요소-->
        21        <li>Milk</li>   <!--p 요소의 형제 요소 중에 p 요소 뒤에 위치하는 ul 요소-->
        22      </ul>
        23    
        24      <h2>Another list</h2>
        25      <ul>
        26        <li>Coffee</li> <!--p 요소의 형제 요소 중에 p 요소 뒤에 위치하는 ul 요소-->
        27        <li>Tea</li>    <!--p 요소의 형제 요소 중에 p 요소 뒤에 위치하는 ul 요소-->
        28        <li>Milk</li>   <!--p 요소의 형제 요소 중에 p 요소 뒤에 위치하는 ul 요소-->
        29    </ul>
        30  </body>
        31</html>
        • 직계 자손 선택자: 바로 하위에 있는 요소 선택 시 사용, 여러 단계에 있는 자손을 모두 선택하는 하위 선택자와는 다름. 꺽쇠로 표현
        1<!DOCTYPE html>
        2<html>
        3<head>
        4  <style>
        5    /* div 요소의 자식요소 중 p 요소 */
        6    div > p { color: red; }
        7  </style>
        8</head>
        9<body>
        10  <h1>Heading</h1>
        11  <div>
        12    <p>paragraph 1</p>  <!--div 요소의 자식요소 중 p 요소-->
        13    <p>paragraph 2</p>  <!--div 요소의 자식요소 중 p 요소-->
        14    <span><p>paragraph 3</p></span> <!--div요소의 자식요소지만 p요소가 아님-->
        15  </div>
        16<p>paragraph 4</p>  <!--p요소지만 div 요소의 자식요소가 아님--
        17</body>
        18</html>
        • 인접 형제 선택자: 특정 요소 바로 다음에 오는 형제 관계 요소 선택.+로 표현
        1<!DOCTYPE html>

  • 의사 클래스
    • 가짜 또는 모조 클래스로 클래스의 특징은 가진다.
    • 형식: :(콜론)의사 클래스 명
    • 링크와 관련된 의사 클래스
      • : link 링크 의사 클래스: 한 번도 방문하지 않은 링크
      • : visited 의사 클래스: 방문한 링크에 CSS 스타일 적용 시 사용
    • 동적 의사 클래스: 마우스와 커서에 관한 상태를 의사 클래스로 나타냄
      • active의사 클래스: 마우스로 클릭 했을 때의 상태 의미
      • hover 의사 클래스: 마우스 커서가 올라간 상태 의미
      • focus 의사 클래스: 서식 폼과 같은 요소에 마우스가 위치하여 입력 또는 선택 상황 의미
    • 구조적 의사 클래스: HTML 구조에 따른 의사 클래스
      • 형재와 자손 그리고 몇 번째 요소인지로 상태 구분
      • root 의사 클래스: 문서 최상위 요소 의미, 단독으로 사용
      • empty 의사 클래스: 비어 있는 요소 의미
      • only-child 의사 클래스: 형제가 없는 요소
      • only-of-type 의사 클래스: 같은 타입의 형제가 없을 때 사용
      • first-child, last-child 의사 클래스: 첫 번째 자손 요소와 마지막 자손 요소를 의미
      • nth-of-type(n), nth-last-of-type(n): 같은 부모 요소의 자손 요소로 특정 요소의 n번째 요소를 의미한다는 점은 같지만 전자는 위에서부터 세어가고 후자는 아래에서부터 세어감.
    • 기타 의사 클래스
      • :lang() 의사 클래스 : 지정한 언어 속성을 가지는 요소
      • :not() 의사 클래스 : 부정을 의미
    • 의사 엘리먼트: 가짜 요소, 선택자에 따라 기존 요소에 추가로 새로운 요소 정의
      • :first-letter 의사 엘리먼트 : 선택자에서 선택한 요소의 첫 번째 글자를 새 요소로 만듦
      • :first-line 의사 엘리먼트 : 선택자에서 선택한 요소의 첫 번째 줄을 새 요소로 만듦
      • :after, :before 의사 엘리먼트 : 선택자에 의해 선택된 요소 앞뒤에 요소를 만듦
